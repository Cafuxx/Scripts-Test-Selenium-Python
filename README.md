# Selenium Python Tests

Proyecto de práctica de automatización QA usando Selenium y Python.

## Tecnologías utilizadas

- Python
- Selenium WebDriver
- ChromeDriver
- WebDriver Manager

## Objetivo

Practicar automatización de pruebas E2E (End-to-End) sobre la página SauceDemo:

https://www.saucedemo.com/

## Tests implementados

- Login test
- Add product to cart
- Remove product from cart
- Checkout flow
- Cart validation
- Product sorting validation

## Instalación

Clonar repositorio:

```bash
git clone https://github.com/Cafuxx/selenium-python-tests.git
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

## Ejecutar tests

```bash
python tests/login_test.py
```

## Aprendizajes

Este proyecto fue realizado para practicar:

- Selenium WebDriver
- Localizadores (ID, CLASS_NAME, CSS_SELECTOR)
- Assertions
- Navegación automatizada
- Validaciones E2E
- Manejo básico de waits
